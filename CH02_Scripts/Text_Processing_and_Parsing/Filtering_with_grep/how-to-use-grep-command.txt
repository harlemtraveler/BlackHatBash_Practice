Filtering with grep

The grep command is used to extract any line containing a pattern from a file.
The basic syntax is:

grep PATTERN SOURCE_FILE

# Basic search for an IP address in a file named "log.txt":
grep "35.237.4.214" log.txt

# The backslash-pipe (\|) as an OR condition:
grep "35.237.4.214\|13.66.139.0" log.txt

# Alternatively, use the -e argument to filter multiple patterns
grep -e "35.237.4.214" -e "13.66.139.0" log.txt

# The pipe (|) provides one command's output as input to another:
ps | grep TTY

# The grep command is case-sensitive. Make search case-insensitive using -i flag:
ps | grep -i tty

# Use the -v argument to exclude lines containing a certain pattern:
grep -v "13.66.139.0" log.txt

# To print only the matched pattern, and not the entire line, use the -o argument:
grep -o "13.66.139.0" log.txt
