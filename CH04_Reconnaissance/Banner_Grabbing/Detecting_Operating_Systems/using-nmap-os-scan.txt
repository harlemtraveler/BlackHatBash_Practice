Using Nmap Operating System Scan

Nmap can guess a target server's operating system by using TCP/IP fingerprinting.
The technique Nmap uses is to identify the OS's implementation the TCP/IP stack
by crafting packets in various ways and analyzing the returned responses.
Each operating system (e.g., Linux, Mac, Windows) implements TCP/IP slightly different.
Nmap analyzes these subtle differences to identify the system.

You can run a Operating System detection scan by using the "-O" flag.
***NOTE: This particular scan requires sudo (root) privileges!

# Run the OS detection scan with the "-O" flag
sudo nmap -O -iL target-hosts.txt
